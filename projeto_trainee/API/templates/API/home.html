Bem vindos ao projeto trainee!

O que fazer no projeto?

O objetivo do projeto é fazer um site/app/web-app (qualquer um da sua escolha) que utilize um API de pagamentos. Com essa API, vocês irão movimentar dinheiro da conta de vocês por meio de pagamentos via QR Code!

O objetivo é ler o contéudo de cada pagamento pelo seu celular e depois inserir os dados que você pegou no banco de dados que irei disonibilizar

Depois disso, o objetivo é mostrar, em uma ou mais páginas, quanto tem a sua conta, o histórico de gastos e os nomes/dados bancários da sua conta

O que eu posso usar?
