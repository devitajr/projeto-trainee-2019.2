api/expenses/expense/

Métodos Rest possíveis: post e get

campos de acesso: date(tipo: date), price(tipo: decimal, 10 digits, 2 decimals), descricao_gasto(tipo: char max_length=200), number(integer), id(integer)

api/banks/account/

Métodos Rest possíveis: patch e get

campos de acesso: banco(tipo: char max_length=200), money_in_account(tipo: decimal, 10 digits, 2 decimals), number(integer)

ATENÇÃO:

Apesar de o valor number ser uma chave estrangeira, tal valor não é entendido como uma chave no banco de dados. Por conta disso, evite a utilizar nos métodos post e patch.
